# dev-mojiscript-cli-hello-world

## Description
A POC for mojiscript programming language.

## Tech stack
- nodejs

## Docker stack
- node:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
